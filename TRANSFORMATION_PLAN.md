# 旅游智慧中心仪表板改造计划

## 目标效果分析
根据提供的图片，目标是将当前的仪表板改造成以下效果：
- **整体布局**：9宫格/网格布局，展示多个数据面板
- **中心元素**：中国地图（蓝色风格，带城市标记）
- **周边面板**：各种数据可视化组件（圆形进度表、柱状图、折线图等）
- **色彩主题**：深蓝色背景 + 蓝色边框 + 黄色强调色
- **整体风格**：科技感、现代化的数据可视化仪表板

## 改造步骤

### 第一阶段：HTML结构调整
1. **重新组织内容容器**
   - 将 `.content` 改为网格布局（CSS Grid）
   - 定义 3行 × 5列 的网格系统
   - 调整各面板的 grid-column 和 grid-row 属性

2. **面板重新分配**
   - 左上角：气象环境、游客流量、车流量
   - 左侧：游客满意度、实时游客流量、今日车流量
   - 中心：中国地图（占据中间大区域）
   - 右侧：停车场数据、游客消费占比
   - 底部：景区门票、游客趋势、呼叫数据

### 第二阶段：CSS样式更新
1. **主容器样式**
   - 使用 `display: grid`
   - 设置 `grid-template-columns` 和 `grid-template-rows`
   - 调整间距和padding

2. **面板样式**
   - 统一边框：蓝色渐变边框（#0066cc → #00ccff）
   - 背景：深蓝色半透明（rgba(4, 16, 30, 0.8)）
   - 圆角：8-12px
   - 阴影：蓝色发光效果

3. **地图面板**
   - 设置为中心大面板
   - 使用 `grid-column: 2 / 5` 和 `grid-row: 1 / 3`
   - 增加蓝色发光边框效果

4. **色彩方案**
   - 主背景：#0a1929
   - 面板背景：#041018
   - 边框色：#0066cc 到 #00ccff 的渐变
   - 强调色：#ffcc00（黄色）
   - 文字色：#7eb7cb（浅蓝）

### 第三阶段：响应式调整
1. 确保在不同分辨率下的适配
2. 调整字体大小和间距
3. 优化移动端显示（如需要）

### 第四阶段：动画和交互
1. 保留现有的脉冲动画（城市标记）
2. 添加面板hover效果
3. 增强蓝色发光效果

## 关键CSS变更

### 主要改动
```css
/* 原来的 flex 布局改为 grid */
.content {
    display: grid;
    grid-template-columns: repeat(5, 1fr);
    grid-template-rows: repeat(3, 1fr);
    gap: 0.1rem;
}

/* 地图面板占据中心大区域 */
.content .center {
    grid-column: 2 / 5;
    grid-row: 1 / 3;
    width: auto;
    margin: 0;
}

/* 左侧面板调整 */
.content .left {
    grid-column: 1;
    grid-row: 1 / 4;
    width: auto;
    flex-direction: column;
}

/* 右侧面板调整 */
.content .right {
    grid-column: 5;
    grid-row: 1 / 4;
    width: auto;
}

/* 统一面板样式 */
.item {
    border: 1px solid;
    border-image: linear-gradient(135deg, #0066cc, #00ccff) 1;
    background: rgba(4, 16, 30, 0.8);
    border-radius: 8px;
    box-shadow: 0 0 20px rgba(0, 102, 204, 0.3);
}
```
