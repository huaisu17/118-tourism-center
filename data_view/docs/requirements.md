# 校园食堂智慧数据管理系统 — 需求分析文档

## 1. 项目背景

本项目为数据库课程大作业，基于"智校数据综合展示中心"数据大屏，构建一套完整的 Web 应用系统，实现校园食堂及后勤数据的统一管理与可视化展示。

## 2. 系统目标

- 为校园食堂管理人员提供数据录入、查询、修改、删除的统一平台
- 通过数据大屏实时展示食堂运营关键指标
- 支持多学校/多区域的数据管理
- 满足课程大作业对增删改查及现场演示的要求

## 3. 功能需求

## 4. 非功能需求

- **易用性**：管理后台界面简洁，操作直观
- **性能**：页面响应时间不超过 2 秒
- **兼容性**：支持 Chrome、Firefox、Edge 等主流浏览器
- **安全性**：密码使用 MD5 加密存储，防止 SQL 注入
- **可维护性**：代码结构清晰，注释完整

## 5. 用户角色

| 角色 | 权限 |
|------|------|
| 系统管理员 | 所有模块的增删改查权限 |
| 普通用户 | 查看数据、录入数据，无删除权限 |
