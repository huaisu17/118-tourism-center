# 数据可视化大屏改造方案

## 📋 目标效果
- 大地图居中显示
- 左右各6个数据块（共12个）
- 每个数据块都有标题区域
- 布局对称，以地图为中心分界线

---

## 🎯 改造步骤

### 第一步：规划新布局结构

#### 1.1 整体布局划分
```
┌─────────────────────────────────────────────────────┐
│                    顶部标题栏                          │
├──────────┬─────────────────────┬──────────────────────┤
│          │                     │                      │
│  左侧    │                     │      右侧            │
│  数据块  │     中间地图区域      │      数据块          │
│  (6个)   │                     │      (6个)           │
│          │                     │                      │
└──────────┴─────────────────────┴──────────────────────┘
```

#### 1.2 尺寸规划（基于3840x2160分辨率）
- **顶部标题区**：3840px × 200px
- **左侧数据区**：900px × 1960px（3行2列，每个数据块约 430px × 300px）
- **中间地图区**：2040px × 1960px
- **右侧数据区**：900px × 1960px（3行2列，每个数据块约 430px × 300px）

## 🔧 常见问题解决

### 问题1：布局错位
**解决：** 检查 `#body` 的 `display: flex` 是否生效，确保没有其他CSS覆盖

### 问题2：数据块太小
**解决：** 调整 `.bodyLeft` 和 `.bodyRight` 的 `width` 值

### 问题3：地图不显示
**解决：** 检查 `.main-map` 的高度是否正确，确保 echarts 初始化代码已更新

### 问题4：标题显示不全
**解决：** 增加 `.block-title` 的 `height` 值，或减小字体大小
