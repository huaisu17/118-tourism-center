- 折线图：细线、面积渐变、末端点高亮
- 饼图：发光环、中间大数字、外圈描边

**文件修改：** `static/chart.js`

**主题配置：**
```javascript
const chartTheme = {
```

**更新的图表函数：**

## 技术细节

### 响应式设计
- 使用 CSS Grid 实现三列布局
- 支持不同分辨率的自适应调整
- 卡片间距统一为 16px

### 动画效果
- 卡片 Hover 效果：平滑过渡 0.3s
- 脉冲动画：2s 循环，三层递进
- 发光效果：使用 box-shadow 和 filter

### 颜色系统
- 主色：青蓝 (#35d8ff)
- 强调色：金黄 (#f7c35f)
- 风险色：红色 (#ff4757)
- 背景：深蓝渐变

### 字体和排版
- 标题：16px，青蓝色，600 字重
- 正文：12-14px，浅蓝灰色
- 数字：14-20px，金黄色，加粗
